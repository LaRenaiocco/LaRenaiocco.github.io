How does a circus clown end up as a software engineer? I'm so glad you asked! I've always been a creative soul and never realized how much creativity there is in coding until I tried it! After pursuing theatre in college and attending the San Francisco Clown Conservatory I achieved my lifelong dream of working professionally in entertainment. I toured full-time for 8 years with Ringling Bros. and Barnum & Bailey Circus in the U.S. and Mexico and then Kinoshita Circus in Japan. But travelling 50+ weeks out of the year can wear down even the most seasoned traveller and I found myself ready to settle down in one place and begin new adventures.

But what to do next with my life? Enter coding! During the pandemic I suddenly found myself with lots of extra time and was finally able dive into software development full-time. The pandemic has been hard for a lot of us, but it allowed me the time to attend Hackbright Academy and really focus my energy on engineering. I'm very excited about my future career in tech and to share my journey with other aspiring devs from non-traditional backgrounds.

Projects

Adventure Awaits

http://adventure-awaits.fun | Github

Plan the perfect adventure with your friends. Adventure Awaits is a full stack web app that allows you, the user, to create itineraries for trips, link those itineraries with your travel companions and collaborate to plan the perfect getaway! Adventure Awaits uses the Google Maps Javascript API as well as Google Places, allowing the user to search for and add destinations such as restaurants, museums and more to the trip. I also used the Twilio API to allow users to get text updates when their friends add activities to the trip!

Tech Stack: Python, JavaScript, HTML/CSS, SQL, Flask, jQuery, Bootstrap, Google Maps, Twilio, AWS Lightsail


Community Library

http://52.205.251.141 | Github

Community Library is a full stack web app that I developed with the goal of learning how to work with the Cloudinary image repository API. In the Community Library a user can search through the database of books in their "community" based on titles, authors and genres with the help of a variety of SQLAlchemy queries. Users can also add or delete books from their personal library (and the community database) and request to borrow books from other users with the Twilio API.

Tech Stack: Python, JavaScript, HTML/CSS, SQL, Flask, jQuery, Bootstrap, Cloudinary, Twilio, AWS Lightsail


Rosie's Movie Awards

https://larenaiocco.github.io/my-movie-awards/ | Github

The Shoppie Awards is a front end only web app that integrates with the OMDB API to allow a user to search the movie database for their favorite movies and "nominate" them for awards.

Tech Stack: JavaScript, HTML/CSS, jQuery, Bootstrap, OMDB API, GitHub Pages
